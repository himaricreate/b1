# Sources and Credits

TruthLab is an educational Computer Science project built with open-source tools and original simulated scenario content.

## Software and Libraries

- **Python**: General-purpose programming language used for the application logic.
- **Streamlit**: Open-source framework used to build the interactive web app interface.
- **Pandas**: Open-source data analysis library used to load and organize scenario data.
- **Pillow**: Listed as a dependency for possible image-based media literacy scenarios and future image handling.
- **Matplotlib**: Listed as a dependency for optional future visualization; the current version uses Streamlit's built-in chart tools.

## General Reference Topics

The scenarios and feedback are original educational simulations inspired by widely taught digital literacy and cybersecurity concepts, including:

- Media literacy practices such as checking source credibility, author identity, publication context, and corroborating evidence.
- Fact-checking habits such as looking for primary sources, reputable news coverage, official public agencies, and transparent evidence.
- Phishing and scam prevention guidance such as avoiding unsolicited links, password requests, urgent threats, unexpected attachments, and requests for sensitive information.
- Scholarship scam warning signs such as upfront fees, guaranteed awards, vague eligibility, unofficial accounts, and requests for identity data through direct messages.
- AI-generated content awareness, including the need to verify image/video provenance, context, and confirmation from trusted sources.
- Statistical reasoning concepts such as sample size, baseline rates, chart labels, polling methods, and exaggerated percentage claims.
- Emotional manipulation patterns such as fear, outrage, guilt, urgency, and pressure to share before verifying.

## Scenario Attribution

All scenarios in `data/scenarios.csv` are fictional and were written specifically for TruthLab. They are not copied real social media posts. Any similarity to common online misinformation, scam, or phishing patterns is intentional for educational purposes.

## Limitations

TruthLab does not perfectly detect misinformation. It is a training simulator that helps learners practice recognizing warning signs and deciding when to trust, doubt, report, or verify information.
